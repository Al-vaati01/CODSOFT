# Brew Haven Landing Page

This is the README file for the Brew Haven landing page. The landing page is an HTML document that showcases the features and offerings of Brew Haven, a neighborhood coffee sanctuary. It includes sections such as a hero section, about us section, menu section, gallery section, testimonials section, and contact us section.

## Getting Started

To view the Brew Haven landing page, simply open the `index.html` file in a web browser.

## Features

The Brew Haven landing page includes the following features:

- Responsive design: The landing page is designed to be responsive and adapt to different screen sizes.
- Navigation: The navigation bar allows users to navigate to different sections of the page.
- Hero section: The hero section introduces Brew Haven and invites users to explore the menu.
- About us section: Provides information about Brew Haven, its story, mission, values, and what it offers.
- Menu section: Showcases the menu offerings, including coffee, tea, other beverages, and food.
- Gallery section: Displays images of the interior ambiance, coffee crafting moments, customer connections, and special events and celebrations at Brew Haven.
- Testimonials section: Features testimonials from satisfied customers.
- Contact us section: Allows users to send a message to Brew Haven.

## Dependencies

The Brew Haven landing page relies on the following files:

- `index.css`: The CSS file that styles the landing page.
- `mb.css`: An additional CSS file for mobile-specific styles.
- `index.js`: The JavaScript file that adds interactivity to the landing page.

## Usage

To customize the Brew Haven landing page, you can modify the HTML, CSS, and JavaScript files as needed. You can update the content, styles, and functionality to suit your requirements.
